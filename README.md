# Cats vs Dogs Recommendation System

This is a Java program that generates personalized advertisements based on user preferences.

## What it does
- Reads lists of cat and dog fans from text files.
- Uses keywords to determine user interests.
- Creates personalized ads from templates.
- Outputs the ads to the console.

## Files
- `Main.java`, `MatchingAlgorithm.java`, `PrintAdvertisements.java`
- `catList.txt`, `dogList.txt`, `catAdvertisement.txt`, `dogAdvertisement.txt`, `targetedwords.txt`
- `pom.xml` for dependencies (Jsoup)
